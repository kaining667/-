# 智能血常规检测系统

孩子第一个用Qt做的项目了QAQ

## 版本

Qt版本：6.5.3（注意！此版本没有自带的mysql驱动，因此需要自己配置，配置方法在CSDN上有，简而言之就是下载驱动，安装驱动，修改Cmake）

mysql版本：8.0.36

c++版本：c++17

cmake版本：3.16

## 数据库构建

创建一个名字为war的数据库

然后导入我的.dbf文件就行了

## 检测数据
